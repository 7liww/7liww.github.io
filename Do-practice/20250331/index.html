# 满足异或值的序列个数

# 题目描述

# 代码

# 思路

动态规划，采用列举法进行

  • 初始化一个二维动态规划数组 dp ，其大小为 (n+1) x (m+1) ，并将所有元素初始化为 0。 dp[i][j] 将表示长度为 i 且异或值为 j 的序列数量。空序列的异或值为 0，因此 dp[0][0] 设为 1。
    vector<vector<int>> dp(n + 1, vector<int>(m + 1, 0)); dp[0][0] = 1;

  • 动态规划的核心部分。外层循环遍历序列长度 i ，内两层循环遍历可能的异或值 j 和 k 。如果 j 和 k 异或的结果小于等于 m ，则更新 dp[i][j ^ k] 的值，它是长度为 i 且异或值为 j ^ k 的序列数量。
    `for (int i = 1; i <= n; ++i) {

    for (int j = 0; j <= m; ++j) {
    for (int k = 0; k <= m; ++k) {
        if ((j ^ k) <= m) {
            dp[i][j ^ k] = (dp[i][j ^ k] + dp[i - 1][j]) % MOD;
        }
    }
}

    }`

  • 返回结果。函数返回长度为 n 且异或值为 m 的序列数量。
    return dp[n][m];
