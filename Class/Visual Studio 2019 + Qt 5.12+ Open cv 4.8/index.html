先装 Qt，再装 Qt-vsaddin，最后在 Visual Studio 中配置 Qt

装软件过程省略 —— 此处记得，安装 QT 的时候记得勾选好 msvc_2017 还有相关的东西

# 配置 QT

# 配置系统变量

  1. 点击 “系统”——“系统信息”——“高级系统设置”——“环境变量”：

  2. 打开点击 “新建”，添加上有关 QT 和 OpenCV 的环境变量，显示如最下面几行：

# 添加 QT 扩展及配置

  1. 进入 VS，点击 “扩展”——“管理扩展”：

  2. 进入以后进行搜索 “QT”，显示出 QT 的选型，将第一个下载并添加：

  3. 添加以后，发现扩展处多了一个 “QT VS Tools”，然后点击 "QT Versions“进行 Version 添加：

  4. 点击 “add new Qt version"，然后选择 QT 文件夹下的 msvc2017 进行添加，添加完成后显示如下：

# 属性设置

  1. 在视图处将属性管理器显示出来：“视图”——“其他窗口”——“属性管理器”

  2. 可看出属性管理器下包含两个形式，一个 Debug，一个 Release：

  3. 点击 “解决方案资源管理器”，在项目名上点击鼠标 “右键”：

  4. 点击 “属性”：

  5. 在属性页点击 “配置属性”——“VC++ 目录”，可看到包含目录和库目录，则进行配置：
    配置如下：

  • 包含目录处添加 OpenCV 与 QT：
    E:\OpenCV\opencv\build\include\opencv
    E:\OpenCV\opencv\build\include\opencv2
    E:\OpenCV\opencv\build\include
    D:\QT5.12\5.12.12\msvc2017_64\bin
  • 库目录处添加：
    E:\OpenCV\opencv\build\x64\vc16\lib
  1. 点击 “链接器”——“输入”，进行 OpenCV 的添加：添加内容可在目录 E:\OpenCV\opencv\build\x64\vc16\lib 下找到。
    配置完成后点击应用，然后点击确定。

# 属性表添加

  1. 点击属性管理器，在 debug 和 Release 下，需要用哪个，则在该处右键点击 “添加新项目属性表”。

  2. 显示出如图所示属性表，进行名称和位置编辑后点击添加则可保存。

  3. 之后别的程序再用，可直接点击 “添加现有属性表”，在之前属性表保存的位置中找到，然后添加即可。

# 配置 QT 文件

  1. 在 VS 新建项目，选择 QT，选择 “Qt Widgets Application"：

  2. 新建成功后，目录显示如下：

  3. 由于直接在 VS 打开.ui 文件，会出现闪退及报错情况，则直接修改打开方式：

  • 右键选择 “打开方式”：
  • 点击添加，显示如图所示：
  • 进行程序处选择，点击…：
  • 选择对应的 QT 软件的位置：在 msvc\bin 目录下，找到 designer.exe
  • 添加后显示，进行名称修改：（修改防止和原有名称重复）：
  • 则添加完成后将该方式设置为默认值，点击确定后，双击可打开.ui 文件，进行设计：

至此，配置完成，可进行程序编写及设计
