# 方法一：累乘法

在第一种方法中，我们通过循环遍历每一位数字，并且在每次迭代中将累积的十进制数乘以基数 R，然后加上当前位的数值。这种方法不需要直接计算幂次，而是通过连续乘以基数 R 来逐步增加权重。

这个操作在每次迭代时执行，其中 number 是累积的十进制结果， base 是基数，而 value 是当前位的数值。

# 方法二：直接幂次法

在第二种方法中，我们直接计算每一位数字乘以基数 R 的相应幂次，然后将所有结果相加。

在这个操作中， pow(base, numberStr.length() - 1 - i) 直接计算了基数 R 的幂次，其中 i 是当前位的索引。

# 区别总结：
  • 累乘法：在每次迭代中更新累积的十进制数，通过乘以基数 R 来增加权重。
  • 直接幂次法：在每次迭代中计算当前位的数值乘以基数 R 的幂次，然后将结果累加。
