# 方法一：累乘法

在第一种方法中，我们通过循环遍历每一位数字，并且在每次迭代中将累积的十进制数乘以基数 R，然后加上当前位的数值。这种方法不需要直接计算幂次，而是通过连续乘以基数 R 来逐步增加权重。

这个操作在每次迭代时执行，其中 number 是累积的十进制结果， base 是基数，而 value 是当前位的数值。

# 方法二：直接幂次法

在第二种方法中，我们直接计算每一位数字乘以基数 R 的相应幂次，然后将所有结果相加。

在这个操作中， pow(base, numberStr.length() - 1 - i) 直接计算了基数 R 的幂次，其中 i 是当前位的索引。

# 区别总结：
  • 累乘法：在每次迭代中更新累积的十进制数，通过乘以基数 R 来增加权重。
  • 直接幂次法：在每次迭代中计算当前位的数值乘以基数 R 的幂次，然后将结果累加。

# 二分查找

# 题目

已知一个有序表（-10, -8, -6, -4, -2, 0, 2, 4, 6, 8, 10, 12, 14, 16），进行在中间值位置向下取整的二分查找值为 0 的元素。

# 解析

  • 对于给定的有序表，初始查找范围为整个表，即下标 low=0, high=13。

  • 第一次比较时，中间元素为 2，与目标值 0 进行比较，由于 2 大于 0，所以目标值在左半部分。

  • 更新查找范围为 low=0, high=5。第二次比较时，中间元素为 - 6，与目标值 0 进行比较，由于 - 6 小于 0，所以目标值在右半部分。

  • 更新查找范围为 low=3, high=5。第三次比较时，中间元素为 - 2，由于 - 2 小于 0，所以目标值在右半部分。

  • 更新查找范围为 low=5, high=5。此时 low=high，mid 值与目标值 0 相等，查找成功。
