# python 安装

  1. 下载 python：https://www.python.org/

  2. 下载的 3.13.3 版本，然后进行程序安装，安装的时候记得勾选 And Python 3.1.3 PATH  选项

  3. 然后进行终端查看，输入 python 可以看到对应的版本并进入 python 环境。

  4. 输入 exit() 可以退出 python 环境

  5. 可以输入 pip show pip 查看是否成功安装

  6. 查看 pip 组件是否存在： pip list ，显示版本则说明存在，不需要安装

  • 若不存在则可通过在线升级进行安装：

# pycharm 的安装

  1. 下载：Download PyCharm: The Python IDE for data science and web development by JetBrains
  2. 下载的 Windows 版本（不要选 arch 版本（Linux 发行版）），然后直接安装，选项全部勾选之后安装即可。
  3. 然后重启就 OK。

# 安装程序

  1. 下载 Anaconda：

  • 官网：Download Anaconda Distribution | Anaconda

  • 镜像站： Index of /anaconda/archive/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror
